## Default Permission

Default permissions for the sqlite plugin - allows all database operations

#### This default permission set includes the following:

- `allow-load`
- `allow-execute`
- `allow-execute-transaction`
- `allow-fetch-all`
- `allow-fetch-one`
- `allow-close`
- `allow-close-all`
- `allow-remove`
